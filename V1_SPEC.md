# TooYoung Platform — V1 Specification

## 1. V1 定義
- V1 為一個完整可運作的前後端平台
- 已完成公共層 → 轉介入口 → 第二層核心功能的貫通
- 本版本刻意不包含任何 auth / login / 帳號系統

## 2. V1 包含的功能模組
- 公共層：Home / Features / About / Contact
- 入口流程：CTA 導向 Dashboard / Cases
- 第二層：
  - Dashboard
  - Case 管理
  - Events
  - Symptoms
  - Timeline

## 3. V1 明確不包含的功能
- 使用者帳號
- 登入 / 權限 / 分享
- 個資與隱私控管
- 對外公開資料功能

## 4. V1 使用流程（簡述）
- 使用者進入公共層
- 透過 CTA 進入第二層
- 建立與查看 Case / Events / Timeline

## 5. V1 的版本定位
- 作為後續 V1.x、V2 的基礎版本
- 所有新增與修改皆以本版本為起點
- auth 與帳號系統將於後續版本加入
